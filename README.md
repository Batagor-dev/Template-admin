**Admin Panel Template**

**Fitur Utama**
- Layout admin panel responsif
- Sidebar + Navbar siap pakai
- Tabel data interaktif
- Form modern dengan enhanced select
- Alert & konfirmasi yang user-friendly
- Struktur file rapi dan scalable


***Teknologi yang Digunakan***
Project ini dibangun tanpa framework frontend berat. Full HTML, CSS, dan JavaScript.

**Core**
- Bootstrap 5
Framework utama untuk layout, grid system, komponen UI, dan responsivitas.

**UI & UX**
- Remix Icon
Library icon modern, clean, dan konsisten untuk dashboard.
